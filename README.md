## 🩺 Skin Cancer Detection using CNN – Deep Learning Project

## Project Summary
Skin Cancer Detection is an AI-driven initiative leveraging Convolutional Neural Networks (CNN) to accurately classify skin lesion images as **benign** or **malignant**. This system assists dermatologists by providing early and automated detection of skin cancer, potentially saving lives through timely diagnosis.

The CNN model is trained on a comprehensive dataset of skin lesion images using advanced deep learning techniques for precise image classification. The final model is deployed as a web application, making it accessible for users and healthcare professionals alike.

---

## ✨ Features

- 📸 **Real-time predictions**: Upload skin lesion images and instantly receive classification results.  
- ✅ **High accuracy**: Differentiates between benign and malignant lesions with precision.  
- 💻 **Interactive web interface**: User-friendly design for seamless interaction.  
- 🌐 **Fully deployed**: Accessible via a web application built with Flask and HTML.  

---

- **Programming Language**: Python 🐍

- **Web Deployment**: Flask, HTML, CSS, JavaScript  
- **Tools**: Jupyter Notebook, Git, VS Code  

---

## 🏗 Model Architecture

- **Type**: Convolutional Neural Network (CNN)  
- **Layers**: Multiple convolutional and pooling layers  
- **Activation Functions**: ReLU (hidden layers), Softmax (output layer)  
- **Optimizer**: Adam  
- **Loss Function**: Categorical Crossentropy  
- **Evaluation Metrics**: Accuracy, Confusion Matrix
